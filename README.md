# Voice Agent Panel

A modern, professional CRM-style panel for managing voice agent campaigns with detailed call logs and analytics.

## Features

- **Campaign Management**: Track both incoming and outgoing campaigns
- **Call Logs**: Detailed logs with phone numbers, duration, and timestamps
- **Voice Recordings**: Playback interface for call recordings
- **Transcripts**: Full conversation transcripts for each call
- **Lead Qualification**: Automatic tagging (Show Project, Callback Required)
- **Keyword Extraction**: Budget, Location, Property Type, and more
- **Analytics**: Real-time statistics and performance metrics

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

The application will be available at http://localhost:3000

## Build

```bash
npm run build
```

## Tech Stack

- React 18
- Vite
- Tailwind CSS
- Lucide React Icons

## Structure

- `/src/components` - React components
- `/src/data` - Dummy data for demonstration
- Multi-layer navigation: Dashboard → Campaign List → Call Logs → Call Details
